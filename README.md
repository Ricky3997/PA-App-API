
## Mentor Platform

First install dependencies

### `npm install`

Start local aws dynamodb

### `java -Djava.library.path=./localdb/DynamoDBLocal_lib -jar ./localdb/DynamoDBLocal.jar`

Then develop (at [http://localhost:3000](http://localhost:3000)
)


### `npm start`

Then release 
### `npm run build`
